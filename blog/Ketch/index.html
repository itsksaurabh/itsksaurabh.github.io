I am one of the core maintainers and creators of Ketch project, which is an application delivery framework that makes it extremely easy for developers to deploy and manage applications on Kubernetes using a simple command-line interface. No YAML is required!

I have been mostly working with Go in building and improving Kubernetes Controllers, CRDs and the CLI. I have added some advanced features like Automatic Canary Deployments, CI integration, Istio and Traefik integration, fixing bugs, and so on.

Learn more about Ketch at https://theketch.io
